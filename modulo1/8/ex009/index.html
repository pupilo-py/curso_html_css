Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase temos um termo em ênfase usando a tag <em> (semântica).

Texto Marcado

Podemos criar um texto marcado usando a tag <mark>.

Texto Grande e Texto Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto Deletado

Podemos marcar um texto como deletado para indicar que ele deve ser lido, mas desconsiderado.

Texto Inserido

Podemos marcar um texto como inserido para indicar que ele foi adicionado depois.

Texto Sobrescrito

Para inserir coisas tipo x20+3.

Texto Subscrito

Para inserir coisas tipo H2O.
